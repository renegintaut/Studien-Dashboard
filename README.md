# Studien-Dashboard
Studien Dashboard - Projekt: Objektorientierte und funktionale Programmierung Python

Dieses Repository enthält den Python-Prototypen meines Studien-Dashboards
aus dem Modul "Objektorientierte & funktionale Programmierung".

Die Anwendung wurde mit Streamlit umgesetzt und ermöglicht eine Übersicht
über den persönlichen Studienfortschritt (ECTS, Zeitplan, Notendurchschnitt).
